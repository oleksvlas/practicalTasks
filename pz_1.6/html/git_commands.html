Create a Repository

From scratch -- Create a new local repositori

$ git init [project name]

Download from an existing repository

git clone my_url

Observe your Repository

List new or modified filed not yet committed

$ git status

Show the changes to files not yet stated

$ git diff

SHow the changes to stages files

$ git diff --cached

Show all stages and unstaged file changes

$ git diff HEAD

Show the changes between two commit ids

$ git diff commit1 commit2

List the change dates and author for a file

$ git blame [file]

Show the file changes for a commit id and/or file

$ git show [commit]:[file]

Show full change history

$ git log

Show change history for file/drectory including diffs

$ git log -p [file/directory]

Working with Branches

List all branches

$ git branch

List all branches, local remote

$ git branch-av

Swich to a branch, my_branch, and update working directory

$ git checkout my_branch

Create a new branch called new_branch

$ git branch new_branch

Delete the branch called my_branch

$ git branche -d my_branch

Merge branch_a into branch_b

$ git checkout branch_b

$ git mege branch_a

Tag the current commit

$ git tag my_tag

Make a change

Stages the file, ready for commit

$ git add [file]

Stage all changed files, readyfor commit

$ git add

Commit all staged files to versioned history

$ git commit -m "commit message"

Commit all your tracked files to versioned history

$ git commit -am "commit message"

Unstages file, keeping the file changes

$ git reset [file]

Revert everything to the last commit

$ git reset --hard

Synchronize

Get the latest changes from origin(no merge)

$ git fetch

Fetch the latest changes from origin and merge

$ git pull

Fetch the latest charges from origin and rebase

$ git pull --rebase

Push local changes to the origin

$ git push

Finaly!

When in doubt, use git help

$ git command --help
